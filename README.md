# Activity Tracker

## Objective
The objective of this project is to implement front-end interactivity in an Activity Tracker application using JavaScript. Users can add activities, update their status from pending to completed, and view real-time progress dynamically.

The application uses DOM manipulation and event handling to update the user interface instantly without reloading the page, improving user experience and demonstrating dynamic data handling in front-end development.

---

## Problem Focus
In Level-1, the application was a static UI with no interactivity. Users could not update activity status or see dynamic changes.

In Level-2, this is improved by:
- Implementing JavaScript event handling
- Managing state using arrays
- Updating UI dynamically using DOM manipulation

This transformation demonstrates real-time updates and responsive UI behavior.

---

## Application Overview
Activity Tracker is a front-end web application built using HTML, CSS, and JavaScript.

It allows users to:
- Add new activities
- Mark activities as completed
- Track progress in real-time

The application includes a dynamic activity list and a visual progress bar for better user experience.

---

## Data Handling Approach
- Activities are stored in a JavaScript array
- Each activity has:
  - Name
  - Status (Pending / Completed)

- New activities are added dynamically
- Status updates modify the array data
- UI is refreshed using DOM manipulation

Optional: Can be extended using Local Storage for data persistence.

---

## UI Components Design
The UI is designed to be simple, interactive, and user-friendly.

Main components:
- Activity List (displays tasks dynamically)
- Input Field and Add Button
- Status Buttons (mark as completed)
- Progress Bar and Percentage Display

---

## Interactivity Implementation
- Event listeners handle user actions
- Add button inserts new activity
- Complete button updates status
- Completed tasks show strike-through effect
- Progress bar updates dynamically
- No page reload required

---

## Application Logic
- Activities stored in array
- Add → Insert into array
- Complete → Update status
- Calculate completed count
- Display progress percentage
- Update UI instantly
